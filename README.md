# django-course

## Introduction
This is a practice project to explore Django framework. The focus is on building a simple web application.

## Features
- User authentication
- CRUD operations
- RESTful API

## Tech Stack
- Python
- Django
- SQLite

## Usage
Clone this repository and run the following commands:

```bash
cd django-course
python manage.py runserver
```
